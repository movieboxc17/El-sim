Electric Circuit Simulator
==========================
A standalone, fully offline circuit simulator.
Works in any modern browser — desktop, laptop, iPad, iPhone, Android.

HOW TO USE
----------
1. Open circuit_simulator.html in any web browser
   (Safari, Chrome, Firefox, Edge — all work)

2. PLACE COMPONENTS
   - Select a component from the left panel (or top bar on mobile)
   - Tap/click the canvas grid to place it

3. WIRE COMPONENTS
   - Select "Wire" tool
   - Tap/click a start point (snaps to grid)
   - Tap/click an end point to complete the wire

4. CONFIGURE
   - Switch to Select mode (button or press S)
   - Tap a component to see its properties at the bottom
   - Edit voltage, resistance, capacitance etc.
   - Tap a switch to toggle it ON/OFF

5. SIMULATE
   - Press the green ▶ Simulate button
   - View voltage, current, power in the right panel
   - Read the Diagnosis section for issues and tips

COMPONENTS
----------
Wire        — connects components
Battery     — voltage source (editable V)
Resistor    — resistance load (editable Ω)
Light Bulb  — glows when current flows (editable Ω)
LED         — lights up with arrow glow (editable forward voltage)
Switch      — tap to toggle ON/OFF
Capacitor   — capacitance (DC steady-state = open circuit)
Voltmeter   — displays voltage reading
Ammeter     — displays current reading
Ground      — reference ground symbol

KEYBOARD SHORTCUTS (desktop)
-----------------------------
S           — Select mode
P           — Place mode
Delete/Bksp — Delete selected component
Escape      — Cancel wire / deselect

TOUCH / IPAD TIPS
-----------------
- Use the horizontal component bar at the top (below toolbar)
- Tap once to start a wire, tap again to end it
- Long-press is not needed — all actions are single taps
- Pinch-to-zoom is disabled to keep grid interaction precise
